Mini and simple responsive social network development with use of:
-html, 
-css, 
-javascript, 
-nodejs, 
-php, 
-mysql 
and all this on apache web server with local host.

The following was done:

-User registration
-Login and logout
-Setting a profile picture
-Adding posts
-Like posts
-Restricting access for the logged in user
-Restricting access for an unregistered user
-Connection to phpmyadmin database and storage of all user data and post data in the database
